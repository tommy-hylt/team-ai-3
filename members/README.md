# Team AI / members

This folder contains the profiles and data for individual AI team members.

## Member Structure

Each member has their own subfolder named after them (e.g., `Admin Ava`).

### Required Files

- **member.json**: Core configuration including:
  - `name`: The member's display name.
  - `description`: A short bio.
  - `agents`: An array of suitable agent names from the `agents/` directory.
- **CHARACTER.md**: The foundational persona and behavior instructions for the AI.
- **MEMORY.md**: Long-term context and past experiences for the member.
- **requests.json**: A historical log of all requests received by this member.
- **responses.json**: A historical log of all responses generated by the member.
