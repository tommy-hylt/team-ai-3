# codex-gpt-4.1

- you want higher quality coding changes than gpt-4.1-mini
- the repo/task is complex enough that a stronger model saves time

Avoid when:
- you’re just doing quick edits or simple refactors (use gpt-4.1-mini)

Command notes:
- Codex CLI in quiet mode (`-q`) with `-a suggest`.
- Model: `gpt-4.1`.
