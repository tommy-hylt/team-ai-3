# claude-fable-5

**Provider:** Anthropic (via Claude Code CLI)
**Model:** claude-fable-5

**Use this agent when:**
- The task is the hardest reasoning or longest-horizon agentic work you have — beyond what Opus can reliably finish.
- You want the most capable Claude model available, regardless of cost.
- The work can run for many minutes per turn (thinking is always on).

**Avoid when:**
- The task is routine coding or triage — use claude-sonnet-5 or claude-opus-4-8 instead; Fable 5 is priced above Opus-tier.
- You need fast turnaround on a simple request.
