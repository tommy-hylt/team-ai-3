# gemini-2.5-flash

**Provider:** Google (via Gemini CLI)
**Model:** gemini-2.5-flash

**Use this agent when:**
- You want the latest fast and cost-effective model from Google.
- You need fast summarization, extraction, drafting, or coding with a massive context window.
- You want a solid default non-Claude/non-OpenAI provider for general tasks.

**Avoid when:**
- The task requires extreme logical deduction and multi-step planning (use gemini-2.5-pro).
