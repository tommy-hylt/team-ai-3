# codex-gpt-5.6-terra

**Provider:** OpenAI (via Codex CLI)
**Model:** gpt-5.6-terra

**Use this agent when:**
- You want the balanced, everyday-work GPT-5.6 tier — strong general coding and reasoning at a lower cost than Sol.

**Avoid when:**
- The task needs the strongest available reasoning (use gpt-5.6-sol) or is simple/latency-sensitive (use gpt-5.6-luna).
