# agy-3.1-flash-lite

**Provider:** Google (via agy CLI)
**Model:** gemini-3.1-flash-lite

**Use this agent when:**
- You want the cheapest, lowest-latency Gemini option for high-volume traffic.
- Tasks are simple: data extraction, quick triage, lightweight agentic steps.

**Avoid when:**
- The task needs strong reasoning or coding depth — use agy-3.5-flash or agy-3.1-pro instead.
